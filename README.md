# 📊 HR Data Analysis with Python

> A complete Exploratory Data Analysis (EDA) project on Human Resource (HR) data using Python. This project focuses on cleaning employee data, analyzing workforce trends, and creating meaningful visualizations to support HR decision-making.

---

# 📌 Project Overview

Human Resource Analytics helps organizations understand their workforce through data. In this project, an HR dataset is analyzed using Python to discover insights about employee distribution, salaries, performance, hiring trends, and attrition.

The project follows a complete Data Analysis workflow:

- Importing the dataset
- Data Cleaning
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insights

---

# 🎯 Objectives

The main objectives of this project are:

- Understand the employee dataset.
- Clean missing and duplicate values.
- Analyze employee demographics.
- Compare salaries across departments.
- Identify high-paying job roles.
- Study employee performance.
- Analyze employee attrition.
- Visualize hiring trends.
- Generate business insights using charts.

# 🚀 How to Run the Project

### Step 1

Clone the repository

```bash
git clone https://github.com/your-username/HR-Data-Analysis.git
```

### Step 2

Navigate to the project folder

```bash
cd HR-Data-Analysis
```

### Step 3

Install required libraries

```bash
pip install pandas numpy matplotlib seaborn
```

### Step 4

Open Jupyter Notebook

```bash
jupyter notebook
```

### Step 5

Run all cells to generate the analysis and visualizations.
